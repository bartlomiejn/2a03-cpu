# 2a03

NES emulator written in C++. Implements elements of a NES system: Ricoh 2A03 (MOS 6502 w/o decimal mode) CPU and a 2C02 NTSC PPU. Includes SDL2 based graphics and a debug view for inspecting PPU/pattern table state. Can run NROM, controller keys are hardcoded: WSAD - Dpad, I - Start, O - Select, K - Button A, L - Button B.
